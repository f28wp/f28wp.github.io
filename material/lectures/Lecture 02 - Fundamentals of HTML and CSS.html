ID vs Class

• Can only be one ID
• Multiple classes
• ID is more specific
• Element can have both ID and multiple classes
ID is good if you want to be very specific. Each element has a unique ID. For example, a single element on screen. [pause] While a class can be used to identify groups of elements. For example, all of the paragraph tags.
