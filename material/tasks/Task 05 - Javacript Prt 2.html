# Lab - Javascript Prt 2 # Introduction In this lab, you’ll further develop your understanding of Javascript. # 1.1 Features/Experiment Take a look at the JavaScript Content Delivery Network (CDNJS) for various open source projects/resources. For example, the **cookiebanner** project, provides a simple way to add a popup dialog to your website to inform visitors about any data laws (e.g., if you are storing/tracking any information). Type the example 'html' into a file and test out the program. ``` document.body.style.height = '200pt'; var script = document.createElement('script'); script.type = 'text/javascript'; script.src = 'https://cdnjs.cloudflare.com/ajax/libs/cookie-banner/1.2.2/cookiebanner.min.js'; script.id = 'cookiebanner'; script['data-height'] = '20px'; script['data-position'] = 'top'; script['data-message'] = 'We use cookies!'; document.head.appendChild(script); script.onload = function() { console.log('loaded script'); // console.log( initCookieConsent ); } console.log('ready'); ``` For example: - Customize the example to provide extra information - Putting your example on your GitHub page - Edit the cookiebanner visuals (background/color/look) - Read the documentation on the website/visit github repository - … ## Other https://cdnjs.com/ Projects Visit the CDNJS website and explore some of the projects, try and implement them, for example: mermaid.min.js and or leaflet.js ## Summary * Task 1.
